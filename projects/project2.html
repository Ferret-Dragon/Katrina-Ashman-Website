Eye to Eye Project

IEEE Pandemic Challenge

Project Overview

During the pandemic lockdown in 2020, IEEE hosted a challenge where students could work together to design and build solutions to problems that came up as a result of the Covid_19 pandemic. For this project, I worked on building an eye tracking software to simulate authentic eye contact during video conferences.

The process of building this project took place over the course of a couple years during which I had the opportunity to work with engineers from IEEE and to learn about various kinds of software, image detection techniques, and machine learning algorithms. This project would become the inspiration for me to further pursue software engineering in years to come.

Technologies Used

Throughout this project, I got to learn about a lot of different computer languages, object detection techniques, machine learning libraries, and algorithms used for object detection. Below are some of the kinds of technology that were used.

  • Python
  • Java
  • JavaScript
  • OpenCV Library
  • Haar Cascades
  • Numpy arrays
  • SQL Databases
