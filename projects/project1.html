Project Title One

Project Overview

During the fall 2025 semester, I got the opportunity to work with a team in a software engineering class to create a website to help with a sustainability issue. Our team chose to build a website for borrowing and lending clothes using the Django framework and Amazon Web Services (AWS).

Our website has the capability of allowing users to checkout and list clothing items in real time, leave reviews on items, and message renters based on the quality of their products with the goal to promote sustainability across college campuses while appealing to students reusable fashion.

Skills Used

  • JavaScript
  • Node.js
  • React
  • Amazon Web Services
  • Django Framework
